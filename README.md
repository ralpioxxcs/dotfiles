# dotfiles
dotfiles is the repository for setting developing environment.

## System requisite
* Linux

## Supported dotfiles
* zsh, oh-my-zsh
* neovim
* tmux
* git

## How to apply
Just execute **config.sh** shell script
```Bash
$ chmod + config.sh
$ ./config.sh
```
